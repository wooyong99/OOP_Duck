## 요구사항
- ```java.awt.Frame```라이브러리를 활용하여 랜덤으로 MallardDuck과 RedDuck을 생성해서 보여주세요.
- MallardDuck은 파란색, RedDuck은 빨간색으로 출력해주세요.
- 랜덤으로 생성된 Duck 객체를 몇 번째로 생성된 객체인지 함께 출력해주세요.
